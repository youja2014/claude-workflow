# Design

Frontend UI/UX 관련 문서를 보관합니다.

## 권장 파일

- `style-guide.md` — 컬러 / 타이포 / 스페이싱 / 그리드
- `interactions.md` — 상호작용 패턴 (네비게이션, 폼, 에러)
- `<screen-or-feature>.md` — 화면 또는 기능별 와이어프레임/컴포넌트 명세

## 룰

- mockup 이미지는 `design/assets/` 에 두고 md 에서 상대 경로로 참조
- 디자인 결정의 *근거*가 필요한 항목은 `docs/decisions/` 에 ADR 로 별도 기록
